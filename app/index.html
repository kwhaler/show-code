Show Code

Jquery plugin showing UI code for Styleguides and Toolkits

View on GitHub Share on Twitter

Example

Meeting and Email Summary

Use

Simply add the class show-code to any container html element. The plugin picks it up, clones and wraps in a collapsible code-block. Then it appends a code icon to the upper right of the container. This allows for an unobtrusive way to share the UI code.
